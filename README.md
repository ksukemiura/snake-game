# snake-game

Classic Snake implemented with plain HTML/CSS/JS.

## Run

Open `index.html` in a browser.

If you want a local server:

```bash
python3 -m http.server
```

Then visit `http://localhost:8000`.

## Controls

- Arrow keys or WASD to move
- Space to pause
- Buttons for Start/Pause/Restart
- On-screen arrows for touch
